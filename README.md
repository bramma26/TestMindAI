# TestMind AI 🚀

An AI-powered test case generation platform built using FastAPI, React, SQLite, and Gemini API.

## Features

- AI Test Case Generation
- Search History
- Delete Records
- Clear History
- Export Excel
- Export PDF
- Copy Test Cases
- Toast Notifications
- Dark Mode (Persistent)
- Dashboard Statistics
- Responsive UI
- Automated API Tests with Pytest

## Tech Stack

### Backend
- FastAPI
- SQLAlchemy
- SQLite
- Gemini API
- Pandas
- ReportLab
- Pytest

### Frontend
- React
- Vite
- Axios
- React Toastify
