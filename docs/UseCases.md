# TestMind AI - Use Cases

## UC-001: Generate Test Cases

### Actor

Tester

### Precondition

User has opened the application.

### Steps

1. Enter requirement or user story.
2. Click Generate.
3. System sends the requirement to AI engine.
4. AI engine generates test scenarios and test cases.
5. System displays the output.

### Postcondition

Generated test scenarios and test cases are displayed.

---

## UC-002: View History

### Actor

Tester

### Precondition

Previously generated results exist.

### Steps

1. Open History page.
2. System fetches saved results.
3. User views previous generations.

### Postcondition

History is displayed.

---

## UC-003: Export Test Cases

### Actor

Tester

### Precondition

Generated test cases are available.

### Steps

1. Click Export.
2. System generates Excel file.
3. File is downloaded.

### Postcondition

Excel file is available to the user.
